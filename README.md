# RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using:

- Sentence Transformers
- ChromaDB
- Hugging Face Transformers
- PDF Retrieval
- Semantic Search

## Features

- PDF ingestion
- Semantic retrieval
- Vector database storage
- Context-aware answer generation
- Source citation display

## Tech Stack

- Python
- ChromaDB
- Sentence Transformers
- Hugging Face
